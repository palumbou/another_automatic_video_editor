Inserisci qui il tuo file musicale royalty-free o con licenza.

Esempi:
- track.mp3
- track.wav
- track.m4a

Se questa cartella è vuota, la pipeline genererà una traccia "musicale" placeholder (sintetica con onde sinusoidali/rumore) per evitare problemi di copyright, ma NON suonerà come una vera colonna sonora.

Suggerimento: per contenuti YouTube, considera l'uso della YouTube Audio Library o altre fonti con licenza appropriata.
